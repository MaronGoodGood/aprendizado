Tags

  • <table>

    • É uma tag para criar tabelas:

    Coluna 1 Coluna 2
    Coluna 3 Coluna 4

  • <del>

    • Reprenta uma parte que foi excluída de um documento:

    Parte que foi excluída.

  • <ins>

    • Representa um texto que foi adicionado ao documento:

    Texto que foi adicionado.

  • <sub>

    • Define um texto que foi subscrito:

    H2O

  • <sup>

    • Define um texto que foi sobrescrito:

    23=8

  • <blockquote>

    • Ele indica que o texto que foi incluído é uma citação longa:

    For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

  • <q>

    • Ele indica que o texto dentro da tag é uma pequena citação:

    Deixe pra trás o que não te leva pra frente.

  • <abbr>

    • Esta tag indica uma abreviação:

    HTML

  • <address>

    • Define as informações de contato do autor do documento ou artigo:

    Written by Jon Doe.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA

  • <cite>

    • Define o título de uma obra, como de um livro:

    1984, de George Orwell.

  • <bdo>

    • Define a direcionalidade do texto a trocando:

    Esse é um texto.

  • <code>

    • Define parte de um código de computador, seu couteúdo é exibido na fonte monoespaçada padrão do navegador:

    $name = ['Jamal','Jamal'];

  • <pre>

    • Esta tag é utilizada para representar o texto pré-formatado, assim como digitado no bloco de notas é escrito:

                Esse é um texto
                com espaço.

  • <kdb>

    • Define a entrada do teclado, sendo que seu conteúdo interno é exibido na fonte monoespaçada padrão de navegado:

    Pressione Ctrl+C para copiar o texto.

    Pressione Ctrl+V para colar o texto.
